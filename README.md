# Landing-Page
CSS Flexbox practice project

## Description
This project is a **practice website** to learn and apply **Flexbox** in CSS.  
It uses HTML and CSS to create layouts and organize content. 

## Technologies Used
- HTML
- CSS (Flexbox)

## What I Learned
- How to use **Flexbox** for layout  
- How to align and space elements with Flexbox properties  
- How to structure a simple HTML + CSS project  
- How to add images and captions in a container

## Flexbox Properties Behavior

| flex-direction | justify-content | align-items | align-content |
|----------------|----------------|------------|---------------|
| row            | X              | Y          | Y             |
| column         | Y              | X          | X             |

> **Legend:**  
> - X → main axis alignment  
> - Y → cross axis alignment  

This table helps understand how **Flexbox properties behave depending on the flex-direction**.
